ERROR

Error recurrente que no me permite realizar la debida marcación para dejar evidencias de las solicitudes de los usuarios

Invalid Receive User

CAUSAS

Este error se presenta debido a que el usuario que esta realizando la actividad no se encuentra autorizado en la tabla 1G – pq usuarios-departamentos, para la marcación determinada, en este caso CALL_ALIA1

SOLUCIÓN

Generar un caso en MyIT solicitando autorización el o los usuarios afectados, sobre la marcación correspondiente. Adjuntando el formato correspondiente PQR (MARCACIONES), con las respectivas firmas autorizadoras. No aplica como incidente
