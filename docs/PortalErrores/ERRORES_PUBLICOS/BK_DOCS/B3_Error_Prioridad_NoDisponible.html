Error Conocido: La prioridad no es disponible para la razón de servicio

Este error se identifica por el mensaje “La prioridad no es disponible para la razón de servicio” al intentar abrir una llamada de servicio:

Pasos que sigue el usuario para identificar el error

Menú Principal Sistema de Administración Cable - opc 1 Búsqueda o Consulta del Suscriptor o la Casa:

Pantalla- Menú Principal Sistema de Administración Cable

Búsqueda o Consulta del Suscriptor o la Casa – Se ingresa la cuenta:

Pantalla - Búsqueda o Consulta del Suscriptor o la Casa

Consulta de suscriptores – F23 LLS:

Pantalla – Consulta de suscriptores

Crear llamadas de servicio – Se digitan los siguientes campos: Persona que se comunica, Problema (Descripción de la falla presentada), Razón de llamada

Pantalla – Crear llamadas de servicio

Se identifica el mensaje de error “La prioridad no es disponible para la razón de servicio”

Pasos para solucionar error

Se identifica el Código que genera el error:

Pantalla – Crear llamadas de servicio

Se pide al usuario que de F1 Sobre el campo del Código o Razón de llamada:

Pantalla - Menú SINTOMAS LLAMADAS DE SERVICIO

F1 Razón de llamada - CREAR LLAMADAS DE SERVICIO.

Se Identifica la tabla en la que se almacena la información de estos códigos

Con la pantalla alterna RR (SHIFT – ESC 1) se ingresa nuevamente

Menú Principal Sistema de Administración Cable - Op 7 Parámetros del Sistema

Pantalla - Menú Principal Sistema de Administración Cable

Menú de Parámetros del Sistema – opc 4 Mantener Tablas del Sistema

Pantalla - Menú de Parámetros del Sistema

Se Busca la tabla anteriormente identificada, 2 sobre la Columna Opc, Enter:

Pantalla – Trabajo con tablas de cable

Se busca el código anteriormente identificado que genera el error

Pantalla – Trabajo con tablas de cable

Se ingresa a F19 NOTE

Pantalla – Descripción de constantes y tablas

41 - 42 Prioridad

43 - 43 Tipo de llamada de servicio *** ver la siguiente página ***

44 - 46 puntos de razón / reserva

47 - 47 "Y" significa agregar uno al contador en el mantenimiento de la planta.

48 - 48 "Y" significa que este es un código público

El error se encuentra de la posición 41 a la 48

Para solucionar este error se debe digitar en estos campos los siguientes caracteres y presionar enter:

600001YY

Pantalla – Trabajo con tablas de cable

Se pide al usuario que valide y el sistema ya debe permitir crear la llamada de servicio dejándolo pasar a la pantalla “ESCOGER LLAMADA DE SERVICIO PARA ACTUALIZARLA”:

Pantalla – Escoger llamada de servicio para actualizarla
