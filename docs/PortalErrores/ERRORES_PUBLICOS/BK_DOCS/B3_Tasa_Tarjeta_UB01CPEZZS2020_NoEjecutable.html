ERROR

Tasa de la tarjeta: UB01/CPE/ZZS/2020-02-03 No se pudo encontrar

CAUSAS

Cuando se va a generar una orden de trabajo y arroja el siguiente error “Tasa de la tarjeta: ARMUH/TVC/BOG/2020-03-12 No se pudo encontrar” es porque cada cliente tiene una cuenta que va asociada a una tarifa, en este caso el error se genera porque esta tarifa no tiene la división y comunidad creada según corresponda a la ubicación del cliente

SOLUCIÓN

Crear división y comunidad para asociar la cuenta del cliente
