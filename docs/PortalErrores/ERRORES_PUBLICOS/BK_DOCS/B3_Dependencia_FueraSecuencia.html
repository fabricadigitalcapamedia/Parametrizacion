ERROR

CAUSAS

Este error se origina debido a que el servicio tiene una novedad de dependencia en la tarifa (tiene dependencia o conflicto con un servicio con dependencia)

SOLUCIÓN

En este caso se debe validar como se solicitó configurar ese servicio en la tarifa de la cuenta afectada. Luego de esto; si la operación es retirar el servicio al tener dependencia no dejara eliminarlo, hay que retirar primero la dependencia. Si se va a insertar, si hay otro servicio con dependencia que genere conflicto con este, no lo dejara insertar, se debe colocar dependencia al servicio a insertar o quitársela al que está generando el conflicto. Normalmente el servicio que tienen la novedad es resaltado por el sistema.

Nota: Si al verificar, si la configuración del servicio en la tarifa afectada, esta según la solicitud de creación de esta tarifa, para realizar cualquier modificación, debe ser a través de un caso con las respectivas firmas autorizadoras
