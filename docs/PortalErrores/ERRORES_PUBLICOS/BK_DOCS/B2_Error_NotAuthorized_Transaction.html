ERROR

Not autorized to transaction code

CAUSAS

El Mensaje se genera, cuando el código resaltado está restringido para una selección especial de usuario o usuarios en su tabla correspondiente y el usuario que intenta utilizarlo no está dentro de esta lista. En conclusión, el usuario que intenta realizar el procedimiento no está autorizado para esta tabla

SOLUCIÓN

Realizar solicitud para autorizar al usuario la tabla que presenta la falla
