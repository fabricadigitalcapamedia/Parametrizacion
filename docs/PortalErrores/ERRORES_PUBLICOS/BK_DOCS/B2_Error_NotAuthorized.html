Documentación error: Not autorized to code ***

Visualización de error

Partes variables de este mensaje (Rojo):

Not autorized to code *** - Not authorized to this table value ***

El Mensaje se genera, cuando el código resaltado está restringido para una selección especial de usuario o usuarios en su tabla correspondiente y el usuario que intenta utilizarlo no está dentro de esta lista.

Solución

El cliente debe solicitar o haber solicitado anteriormente la autorización a estos servicios por medio de un debido formato Brieft con las firmas respectivas de gerencia.

Para dar las autorizaciones respectivas se indaga primeramente la tabla donde está alojada el servicio ubicándose donde se digito el código y presionando la tecla F1, esto despegara una ventana emergente que dejara en visto la tabla :

Cuando ya se a identificado la tabla, se ubica en el menú principal, opción 7 – opción 4, buscamos la tabla correspondiente y entramos con la opción 2

Buscamos el código correspondiente en la opción SCAN FOR y colocamos en el código la opción 8 AUTHORITY:

En esta opción ingresamos el usuario correspondiente que solicito o solicita los permisos sobre este código:

Finalmente se reporta la corrección de la anomalia
